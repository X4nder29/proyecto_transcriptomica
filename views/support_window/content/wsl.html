Para ejecutar TranscriptoHub en Windows es necesario tener instalado en el sistema WSL (Subsistema de Linux en Windows). Para ello, sigue estos pasos:

3.1 Abrir PowerShell como Administrador

3.2 Habilitar WSL


dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

3.3 Habilitar Máquina Virtual


dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

3.4 Reiniciar el Equipo

  1. Cierra todas las aplicaciones.
  2. Reinicia Windows.

3.5 Desinstalar e Instalar TranscriptoHub

Una vez reiniciado el equipo es necesario desinstalar TranscriptoHub y volver a instalar para así tener una instalación correcta del software en su totalidad.

  1. Abre el menú Inicio y busca “TranscriptoHub”.
  2. Haz clic derecho y selecciona Desinstalar.
  3. En la ventana del sistema, confirma la eliminación.
