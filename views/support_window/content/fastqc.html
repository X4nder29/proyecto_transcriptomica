3.2.2 FastQC — Panel de control de calidad

¿Qué es este panel?

El módulo FastQC dentro de la Main Window permite generar y visualizar informes de control de calidad para archivos FASTQ. Desde aquí se selecciona el archivo a analizar, se visualiza la información básica del fichero y se exploran las gráficas que FastQC genera (calidad por base, contenido GC, duplicaciones, adaptadores, etc.). Además cuenta con dos botones en la esquina superior derecha para acceder rápidamente al manual del panel y para ver el comando exacto que se ejecutó para generar el informe.

Vista general del panel

  • Selector de archivo (arriba izquierda): área donde se selecciona el archivo FASTQ sobre el que se ejecutará FastQC.

  • Cabecera de informe (a la derecha del selector): datos básicos extraídos del informe (File Name, File Type, Encoding, Total Sequences, Sequences Flagged as Poor Quality, Sequence Length, %GC). Esta cabecera aparece cuando ya existe un informe generado para el archivo seleccionado.

  • Resumen (columna izquierda, debajo del selector): lista de entradas que corresponden a las gráficas/informes parciales (por ejemplo: Calidad por base, Puntuaciones por secuencia, Contenido GC, Distribución de longitud, N por base, Duplicaciones, Secuencias sobre-representadas, Contenido de adaptador). Al hacer clic en una entrada de la lista, la gráfica correspondiente se muestra en el área principal a la derecha.

  • Área de gráfica / visualización (derecha): muestra la gráfica seleccionada desde el resumen; si no hay informe o no se ha seleccionado una gráfica, muestra un mensaje indicándolo.

  • Botones superiores (esquina superior derecha): dos accesos rápidos:

    • Botón Manual — abre la sección correspondiente del manual en la Support Window.

    • Botón Ver comando — abre una ventana modal que muestra la línea de comando exacta que se ejecutó para generar el informe FastQC (ruta al ejecutable / script y parámetros).

Flujo de uso (paso a paso)

  1. Seleccionar archivo

    • En el área Archivo de entrada (arriba a la izquierda) haga clic y seleccione el archivo FASTQ desde el workspace.

    • Nota: sin seleccionar archivo el panel permanece en estado vacío y no muestra cabecera ni gráficos (ver figura de estado vacío).

  2. Generar / consultar informe

    • Si ya existe un informe FastQC para ese archivo, la cabecera con la información básica aparecerá y la lista de Resumen se habilitará.

    • Para ver la línea de comando que se ejecutó (útil para reproducibilidad o depuración) pulse el botón Ver comando en la esquina superior derecha. Se abrirá una ventana modal con la instrucción exacta (ruta al script .bat/.sh, parámetros y rutas de entrada/salida). (Ver figura del comando.)

  3. Explorar las gráficas

    • En la columna Resumen (izquierda) haga clic sobre cualquiera de los elementos (por ejemplo “Calidad de secuencia por base”). La gráfica correspondiente se mostrará en el área principal derecha.

    • Para comparar resultados, seleccione diferentes elementos del resumen y observe cómo cambian las visualizaciones.

  4. Interpretar la cabecera

    • La cabecera contiene métricas clave del archivo analizado: tipo de archivo, encodificación detectada, número total de secuencias, tamaño de secuencias señalado como de baja calidad y %GC. Úselas para decidir si continuar con recorte o filtrado (Trimmomatic / SortMeRNA).

Comportamientos y estados importantes

  • Estado vacío: cuando no hay archivo seleccionado, el panel muestra un mensaje que indica seleccionar un informe para visualizarlo y la lista de resúmenes aparece desactivada. (Ver figura estado vacío).

  • Reporte disponible: al seleccionar un archivo con informe disponible, se habilita la cabecera y la lista de resúmenes; las gráficas se cargan al seleccionar cada ítem. (Ver figura reporte seleccionado y figura con gráfica mostrada).

  • Ver línea de comando: la ventana de comando muestra la llamada exacta a FastQC (o al script que envuelve a FastQC) y las rutas utilizadas; es útil para reproducir el paso manualmente o para documentar el proceso en metadatos del proyecto. (Ver figura comando).

  • Acceso al manual: el botón Manual abre la Support Window en la sección dedicada a FastQC para consultar en el contexto del panel las instrucciones y explicaciones sin salir de la aplicación.

Consejos prácticos

  • Antes y después: ejecute FastQC antes y después de recortar (Trimmomatic) para verificar la mejora en la calidad.

  • Usa la cabecera para decisiones rápidas: si la columna “Sequences Flagged as Poor Quality” o el %GC son inusuales, considere ajustar parámetros de recorte/filtrado.

  • Documenta los comandos: guarde o copie la línea de comando mostrada para reproducibilidad del análisis o inclusión en el informe final.

Notas

  • Al finalizar la generación del reporte se quedará un tiempo cargando, más de lo normal, y parecerá que el software dejó de responder pero no es así, la última etapa de la generación de reporte toma algo más de tiempo.
