Configuración

El Panel de Configuración puedes verificar que todos los programas y dependencias requeridos por TranscriptoHub estén correctamente instalados. También puedes consultar sus rutas de instalación y acceder directamente al manual de usuario.


Vista general



Manual de Usuario

En la parte superior del panel se encuentra un botón para abrir directamente este manual desde la aplicación.

Botón de acceso al manual de usuario


Programas Instalados

Se muestra una lista con los cuatro programas principales de análisis, junto con su estado (Instalado / Instalar) y la ruta desde donde se ejecutan.

Programa Estado Ruta de Instalación

Trimmomatic Instalado .../Trimmomatic/trimmomatic_0.39.jar
SortMeRNA Instalado .../SortMeRNA/bin/sortmerna.exe

Dependencias

En esta sección puedes verificar si las herramientas externas requeridas están disponibles en el sistema. Estas no se instalan automáticamente, pero el software detecta si están presentes:

Dependencia Estado Función
Java Instalado Necesario para ejecutar Trimmomatic
WSL Instalado Requerido para ejecutar herramientas Linux
Krona No instalado Utilizado para visualizar taxonomías

Dependencias del sistema


Este panel no permite editar rutas ni modificar la instalación: su función es únicamente informativa, para asegurar que todos los componentes estén listos antes de iniciar el análisis.
